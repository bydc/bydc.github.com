CF1458E Nim Shortcuts

发布于 2020-12-20 | 12分钟 | 2141字数

题意

题意简述：两堆石子，先手后手轮流在一堆中取石子，不能不取。题目给出nnn种两堆石头的状态(xi,yi)(x_i,y_i)(xi​,yi​)，当一个人取石头之前，两堆石头的状态为(0,0)(0,0)(0,0)或者以上nnn种状态，这个人输了。
假设两者都足够聪明，qqq次询问ai,bia_i,b_iai​,bi​表示初始两堆石子的数量为ai,bia_i,b_iai​,bi​，先手必胜还是必败，前者输出WIN后者输出LOSE。
数据范围：n≤105,1≤xi,yi,ai,bi≤109n \le 10^5,1 \le x_i,y_i,a_i,b_i \le 10^9n≤105,1≤xi​,yi​,ai​,bi​≤109。

题解

一些约定：
两堆石子数量分别为x,y，下文称(x,y)(x,y)(x,y)。
若(x,y)(x,y)(x,y)被banbanban掉了，那么称(x,y)=−1(x,y)=-1(x,y)=−1
若(x,y)(x,y)(x,y)先手必胜，那么称(x,y)=0(x,y)=0(x,y)=0
若(x,y)(x,y)(x,y)先手必败，那么称(x,y)=1(x,y)=1(x,y)=1

可以发现对于所有(x,y)=1(x,y)=1(x,y)=1都不存在(x,z)=1(x,z)=1(x,z)=1或(z,y)=1(z,y)=1(z,y)=1，即每行每列最多一个111。
首先m=0m=0m=0时，LOSE等价于x=yx=yx=y，即先手无论取多少，后手都在另一堆里取相同多的。
然后考虑m=1m=1m=1。对于每个(x,y)=−1(x,y)=-1(x,y)=−1，分别考虑对于−1,0,1-1,0,1−1,0,1的分布的影响。

  • −1-1−1：(x,y)=−1(x,y)=-1(x,y)=−1
  • 000：
    1. x=yx=yx=y，对于zzz，(z,y)=0,(x,z)=0(z,y)=0,(x,z)=0(z,y)=0,(x,z)=0，证明同第2,3种情况。
    2. x>yx>yx>y，对于zzz，(x,z)=0(x,z)=0(x,z)=0，考虑第xxx行原本唯一的111：(x,x)(x,x)(x,x)，先手可以取到(x,y)(x,y)(x,y)，后手就输了。因此(x,x)=0(x,x)=0(x,x)=0
    3. x<yx<yx<y，对于zzz，(z,y)=0(z,y)=0(z,y)=0，考虑第yyy列原本唯一的111：(y,y)(y,y)(y,y)，先手可以去到(x,y)(x,y)(x,y)，后手就输了。因此(y,y)=0(y,y)=0(y,y)=0
  • 111：
    1. x=yx=yx=y，对于z≥xz\ge xz≥x，原本(z,z)=1(z,z)=1(z,z)=1，变成(z+1,z+1)=1(z+1,z+1)=1(z+1,z+1)=1。即删掉第xxx行，第yyy列后，111的分布不变
    2. x>yx>yx>y，对于z≥xz\ge xz≥x，原本(z,z)=1(z,z)=1(z,z)=1，变成(z+1,z)=1(z+1,z)=1(z+1,z)=1。即删掉第xxx行后，111的分布不变
    3. x<yx<yx<y，对于z≥yz\ge yz≥y，原本(z,z)=1(z,z)=1(z,z)=1，变成(z,z+1)=1(z,z+1)=1(z,z+1)=1。即删掉第yyy列后，111的分布不变

可以发现−1-1−1很好维护。000为当前点被删了或者删掉后的(x,y)(x,y)(x,y)中x≠yx\neq yx​=y，111为当前点没有被删并且删掉后的(x,y)(x,y)(x,y)中x=yx=yx=y。
因此关键在于，求出哪些行或列被删了。
接着考虑m>1m>1m>1的情况。因为已经会m=1m=1m=1，因此可以用某种顺序来依次用m=1m=1m=1的方法考虑，其核心是后面的点不会影响前面的点。
发现一个点(x,y)(x,y)(x,y)会影响到max⁡(x,y)\max(x,y)max(x,y)行或列，因此根据max⁡(x,y)\max(x,y)max(x,y)的从小到大维护，但需要动态删一行或一列。

抽象成一个数据结构问题：有nnn个二元组(xi,yi)(x_i,y_i)(xi​,yi​)，一个二元组(xi,yi)(x_i,y_i)(xi​,yi​)的权值为max⁡(xi,yi)\max(x_i,y_i)max(xi​,yi​)，要支持三种操作：

  1. 查询权值最小的二元组
  2. 删除权值最小的二元组
  3. 给所有二元组的xix_ixi​或yiy_iyi​减111
    考虑维护两个tag0,tag1tag0,tag1tag0,tag1表示xix_ixi​减少了多少，yiy_iyi​减少了多少，令delta=tag0−tag1delta=tag0-tag1delta=tag0−tag1。
  • 初始状态为x−y=deltax-y=deltax−y=delta 的二元组(x,y)(x,y)(x,y)当前状态时x′=y′x'=y'x′=y′
  • 初始状态为x−y>deltax-y>deltax−y>delta 的二元组(x,y)(x,y)(x,y)当前状态时x′>y′x'>y'x′>y′
  • 初始状态为x−y<deltax-y<deltax−y<delta 的二元组(x,y)(x,y)(x,y)当前状态时x′<y′x'<y'x′<y′

可以发现将所有二元组按照xi−yix_i-y_ixi​−yi​排序后，所有xi≥yix_i \ge y_ixi​≥yi​的二元组始终是一个前缀，xi≤yix_i \le y_ixi​≤yi​的二元组始终是一个后缀。因此可以分别计算xi≥yix_i \ge y_ixi​≥yi​的二元组最小值和xi≤yix_i \le y_ixi​≤yi​的二元组最小值。前后缀min⁡\minmin，单点修改可以使用线段树，注意一行一列不能被删多遍。

const int MAXN = 1e5 + 10;
struct Node{
	int x,y;
}a[MAXN];
int n,m;

inl bool cmp(Node a,Node b){
	return a.x-a.y>b.x-b.y;
}
struct Segment{
	#define ls rt<<1
	#define rs rt<<1|1
	pii Min[MAXN<<2];
	int tag;
	inl void pushup(int rt){
		Min[rt]=min(Min[ls],Min[rs]);
	}
	inl pii query(int rt,int l,int r,int L,int R){
		if(L>R) return {2e9,-1};
		if(l>=L && r<=R) return Min[rt]; 
		int mid=l+r>>1;
		if(R<=mid) return query(ls,l,mid,L,R);
		else if(L>mid) return query(rs,mid+1,r,L,R);
		else return min(query(ls,l,mid,L,R),query(rs,mid+1,r,L,R));
	}
	inl void update(int rt,int l,int r,int pos) {
		if(l==r) return Min[rt]={2e9,-1},void();
		int mid=l+r>>1;
		if(pos<=mid) update(ls,l,mid,pos);
		else update(rs,mid+1,r,pos);
		pushup(rt);
	}
	inl void build(int rt,int l,int r,int opt){
		if(l==r) {
			Min[rt]={opt?a[l].y:a[l].x,l};
			return ;
		}
		int mid=l+r>>1;
		build(ls,l,mid,opt),build(rs,mid+1,r,opt);
		pushup(rt);
	}
	#undef ls
	#undef rs
}t[2];
int main(){
	//freopen("in.txt","r",stdin);
	read(n),read(m);
	rep(i,1,n) read(a[i].x),read(a[i].y),Map[{a[i].x,a[i].y}]=1;
	sort(a+1,a+n+1,cmp);
	per(i,n,1) Set.push_back(a[i].x-a[i].y);
	t[0].build(1,1,n,0),t[1].build(1,1,n,1);
	int delta=0;S.push_back(-1),T.push_back(-1);
	rep(i,1,n) {
		int pos=n-(lower_bound(all(Set),delta)-Set.begin());
		pii lans=t[0].query(1,1,n,1,pos-1),rans=t[1].query(1,1,n,pos,n);
		lans.fi-=t[0].tag,rans.fi-=t[1].tag;
		pii Min=min(lans,rans);
		int id=Min.se;
		if(a[id].x-a[id].y==delta) {
			if(S.back()!=a[id].x && T.back()!=a[id].y) {
				t[0].tag++,S.push_back(a[id].x);
				t[1].tag++,T.push_back(a[id].y);
			}
		} else if(a[id].x-a[id].y>delta) {
			if(S.back()!=a[id].x) delta++,t[0].tag++,S.push_back(a[id].x);
		} else if(a[id].x-a[id].y<delta) {
			if(T.back()!=a[id].y) delta--,t[1].tag++,T.push_back(a[id].y);
		}
		t[0].update(1,1,n,id);t[1].update(1,1,n,id);
 	}
 	rep(i,1,m) {
 		int x,y;
 		read(x),read(y);
 		if(Map[{x,y}]) {puts("LOSE");continue;}
 		if(binary_search(all(S),x) || binary_search(all(T),y)) {puts("WIN");continue;}
		x-=lower_bound(all(S),x)-S.begin();
 		y-=lower_bound(all(T),y)-T.begin();
 		puts(x==y?"LOSE":"WIN");
	}
	return 0;
}
